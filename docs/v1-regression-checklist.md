# 第一版主流程回归清单

## 目标
确认第二版基础设施接入后，第一版已上线主流程仍然保持可用。

## 核心流程

1. 从零创建项目
2. 导入旧 Skill
3. 生成草稿
4. 预览内容
5. 导出 ZIP
6. 备份导出
7. 备份恢复

## 检查项

### 1. 从零创建项目
- 首页输入目标
- 点击“从零创建”
- 成功进入创建流程

通过标准：
- 新项目创建成功
- 目标被带入项目

### 2. 导入旧 Skill
- 首页进入导入模式
- 粘贴旧 Skill 文本
- 触发提取

通过标准：
- 项目进入导入模式
- 主要字段可被提取

### 3. 生成草稿
- 补齐目标、任务、输入、输出
- 点击生成草稿

通过标准：
- `draft` 成功生成
- 说明版和示例内容存在

### 4. 预览内容
- 查看说明版
- 查看 `SKILL.md`
- 查看示例结果

通过标准：
- 三种预览都可显示

### 5. 导出 ZIP
- 点击导出

通过标准：
- ZIP 成功下载
- 导出文件名正常

### 6. 备份导出
- 在“我的项目”里下载备份

通过标准：
- 备份文件成功生成

### 7. 备份恢复
- 导入上一步备份文件

通过标准：
- 项目列表成功恢复
- 当前项目可继续编辑
